## Usage

Use Skeleton to create a placeholder for loading content. Component support following props:

- `height` – height in px (number) or string for other css values
- `width` – width in px (number) or string for other css values
- `radius` – value from `theme.radius` or number to set border-radius in px
- `circle` – if true width, height and border-radius will equal to value specified in `height` prop

## With content

If you want to indicate loading state of content that is already on page you can wrap it with Skeleton
and control loading overlay visibility with `visible` prop:

## TypeScript

Import component props type:

```tsx
import type { SkeletonProps } from '@mantine/core';
```

`SkeletonProps` extends `React.ComponentPropsWithoutRef<'svg'>`, all other props are available under _Component props_ tab.
